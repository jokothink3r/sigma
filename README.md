# SIGMA Analyzer V3.1 — Advanced Malware & File Security Scanner

SIGMA Analyzer V3.1 adalah alat analisis keamanan tingkat lanjut yang dilengkapi fitur lengkap untuk mendeteksi, menganalisis, dan memantau ancaman digital. Tools ini menyediakan File Analyzer, Folder Analyzer, integrasi VirusTotal, ekstraksi IOC (URLs, IPs, email, phone, dll), analisis PE file, serta Real-Time Protection (RTP) untuk memantau aktivitas berbahaya secara langsung.

Tools ini juga memiliki sistem lisensi PRO, HWID generator, logging otomatis, quarantine system, dan antarmuka CLI interaktif yang mudah digunakan.
